\# Chemical Equipment Parameter Visualizer



This project is developed as part of the FOSSEE Internship Screening Task.  

It is a hybrid application that runs as both a Web App and a Desktop App.



\## Features

\- Upload CSV file

\- Backend processes data using Django

\- Shows summary and charts

\- Web frontend using React + Chart.js

\- Desktop frontend using PyQt5 + Matplotlib

\- Stores last 5 upload summaries using SQLite



\## Tech Stack

\- Backend: Django + Django REST Framework  

\- Web: React.js + Chart.js  

\- Desktop: PyQt5 + Matplotlib  

\- Database: SQLite
